Djoudini Kodi Repository

Installation (einfach):

  1. Lade das Repository-Addon herunter: repository.djoudini-1.0.0.zip
  2. In Kodi: Einstellungen → System → Add-ons → Unbekannte Quellen aktivieren
  3. Add-ons → Aus ZIP installieren → die heruntergeladene repository.djoudini-1.0.0.zip wählen
  4. Add-ons → Aus Repository installieren → Djoudini Repository → Video-Add-ons → Cumination installieren

Direkte Zips (falls nötig):

  • addons.xml
  • addons.xml.md5
  • repository.djoudini-1.0.0.zip (in repo)
  • plugin.video.cumination-1.1.180.zip
  • repository.dobbelina-1.0.4.zip (wird weiterhin gebraucht für resolveurl.xxx)
